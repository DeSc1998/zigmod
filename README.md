- Zig master `0.8.0-dev.1385+3dd8396a5`

### Build from Source
Initially,
```
$ git clone https://github.com/nektro/zigmod --recursive
$ zig build -Dbootstrap
$ ./zig-cache/bin/zigmod fetch
```

To build,
```
$ zig build
$ ./zig-cache/bin/zigmod
```
